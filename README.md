## Tailwind setup

- npm i tailwindcss tailwindcss/vite
- Add tailwindcss() in vite.config.ts
- paste this in App.css : @import "tailwindcss";

## API integration

- Youtube developer (https://developers.google.com/youtube/v3/docs/videos/list#http-request) API's integrated using API key
- Auto suggestion API integration with debouncing
